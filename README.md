🌍 Análisis Global de Datos de Salud con Python
📌 Descripción del Proyecto

Este proyecto consiste en el análisis y procesamiento de datos de salud provenientes de distintos países utilizando Python. El objetivo principal fue estudiar cómo una enfermedad afectó a diferentes regiones del mundo, identificando patrones, tendencias y variaciones entre países.

Para el desarrollo del proyecto se aplicaron técnicas de limpieza, transformación y análisis de datos, así como herramientas de visualización que permiten interpretar la información de manera más clara y dinámica.

🛠️ Herramientas Utilizadas
Python

📊 Características Principales

✅ Limpieza y procesamiento de datos
✅ Análisis estadístico y comparativo
✅ Visualización de información mediante gráficas
✅ Interpretación de tendencias globales de salud

🎯 Objetivo

Este proyecto fue desarrollado con fines académicos para fortalecer habilidades en programación, análisis de datos y visualización de información utilizando Python.
